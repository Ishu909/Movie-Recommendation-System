# 🎬 Movie Recommendation System (Cosine Similarity)

## 📌 Overview

A movie recommendation system that suggests similar movies based on cosine similarity using movie metadata.

---

## 🚀 Features

* Content-based recommendation
* Uses cosine similarity
* Fast and simple predictions
* Works on IMDB Top 1000 dataset

---

## 🧠 Approach

* Data preprocessing (cleaning, feature selection)
* Feature vectorization
* Cosine similarity calculation
* Recommend top similar movies

---

## 📂 Dataset

Dataset used: IMDB Top 1000 Movies

File required:
* imdb_top_1000.csv

---
